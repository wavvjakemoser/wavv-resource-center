# Intercom Help Article Structure Reference
Source: https://help.wavv.com/en/articles/15383103-chrome-extension-beta-setup-guide

## Layout
- Title at top (large H1)
- Tagline/subtitle below title
- "Written by [Author] · Updated [date]" metadata
- Intro paragraph (article overview)
- Numbered sections (H1 headings: "1) Get the Install Link", "2) Open Google Chrome", etc.)
- Each section: heading + body text + optional screenshot (full-width inline image) or Loom video embed

## Table of Contents (right sidebar, sticky)
- Auto-generated from H1 section headings
- Anchor links: e.g. #h_04dc282dba for "1) Get the Install Link"
- Sections listed: 1–9 + "You're live!"
- Clicking jumps to that section on the page

## Media per section
- Screenshots: full-width inline images below section text
- Videos: Loom embeds (embedded player inline, with playback speed controls, comment count, copy link)
- The Loom video appears as a full-width embedded player with controls

## Additional features
- Reaction buttons at bottom (😞 😐 😃)
- Related articles sidebar
- View count shown on Loom embed

## Key structural elements to replicate
1. Sticky table of contents (right side) with anchor jump links
2. Numbered sections as primary navigation unit
3. Per-section: text + optional screenshot OR Loom/video embed
4. Intro paragraph before sections
5. Author + last updated metadata
6. Article title + tagline
